# Microstrip Patch Antenna Design - ANSYS HFSS

> 2.4 GHz WiFi band patch antenna designed and simulated using ANSYS HFSS

## Results Summary
| Metric | Result |
|--------|--------|
| Resonant Frequency | 2.41 GHz |
| Return Loss | -28.5 dB |
| Gain | 6.2 dBi |
| Efficiency | 87% |
| Bandwidth | 48 MHz |

## Files
- antenna_specs.md - Full design calculations and simulation results
- hfss_design_notes.md - HFSS setup, boundary conditions, mesh strategy

## Tools Used
- ANSYS HFSS 2022 R2
- Design method: Transmission Line Model
- Substrate: FR4 (er = 4.4)
